Smart Bookmark · 隐私政策

生效日期: 2025-01-15 · 版本 0.3.0 · 开发者 xiaoniuge36

1. 总则

Smart Bookmark 是一个本地优先的浏览器扩展，不运营任何服务器，不收集任何用户数据，不分析用户行为。本政策解释扩展所使用的权限与数据去向。

2. 我们处理的数据

数据类别用途存储位置是否上传
书签（bookmarks）展示 / 搜索 / 清理 / 画像浏览器本地书签库❌ 永不上传
浏览历史（history）统一搜索浏览器本地历史库❌ 永不上传
常用站点（topSites）新标签页快捷入口仅内存读取❌ 永不上传
扩展设置个性化 UIchrome.storage.local❌ 永不上传
AI API Key（可选）调用你自己的 OpenAI / Anthropic 账户chrome.storage.local❌ 仅发往对应厂商的 API，直连不经我们
AI 对话消息流式显示不落盘➡️ 仅当你主动点发送时发给你选择的 Provider

3. 权限使用说明

权限用途
bookmarks核心功能：清理、画像、搜索
storage保存设置与 API Key（仅本机）
contextMenus右键菜单
sidePanelAlt+B 打开侧边栏
history搜索历史记录
topSites新标签页常用站点
tabs打开新标签页
scripting注入悬浮球（可关闭）
https://*/*失效链接检测（HEAD 探测） + 悬浮球

4. 第三方服务

默认零第三方。仅在你主动启用 AI 助手并填入 API Key 时，扩展从你的浏览器直接向 api.openai.com 或 api.anthropic.com 发起请求。请求内容为你输入的对话文本，不包含书签数据。

5. 数据保留与删除

  • 卸载扩展 → 所有 chrome.storage.local 数据清除
  • 书签由浏览器管理
  • 随时可在"设置"清空 API Key

6. 儿童隐私

本扩展不针对 13 岁以下儿童，亦不会主动收集任何个人数据。

7. 政策变更

通过 GitHub commit 历史公开追踪；不会降低对已有用户的保护。

8. 联系方式

请在 GitHub Issues 反馈。

Privacy Policy

Effective: 2025-01-15 · Version 0.3.0 · Developer: xiaoniuge36

1. Overview

Smart Bookmark is a local-first browser extension. We run no servers, collect no user data, and deploy no analytics. This policy explains permissions and data flows.

2. Data We Process

CategoryPurposeStorageUploaded?
BookmarksDisplay / search / clean / profileBrowser local store❌ Never
HistoryUnified searchBrowser local store❌ Never
Top sitesQuick links on new tabIn-memory❌ Never
SettingsUI personalizationchrome.storage.local❌ Never
AI API Key (optional)Access YOUR OpenAI / Anthropic accountchrome.storage.local❌ Only sent to the chosen vendor, direct from your browser
AI chat messagesStreamed displayNot persisted➡️ Only when you press Send

3. Permissions

PermissionPurpose
bookmarksCore functionality: clean / profile / search
storagePersist settings & key (local only)
contextMenusRight-click actions
sidePanelOpen side panel with Alt+B
historySearch history records
topSitesShow popular sites
tabsOpen new tabs
scriptingInject the floating widget (toggleable)
https://*/*HEAD probes for link liveness + widget

4. Third-Party Services

None by default. Only when you opt in to the AI assistant and provide an API key, the extension sends requests directly from your browser to api.openai.com or api.anthropic.com.

5. Retention & Deletion

  • Uninstall → all local data wiped
  • Bookmarks are managed by the browser
  • Clear your API key anytime in Settings

6. Children's Privacy

Not designed for children under 13. No personal data is intentionally collected.

7. Changes

Changes are tracked publicly via Git commits. Protections already granted will not be reduced.

8. Contact

Please file an issue on GitHub Issues.
